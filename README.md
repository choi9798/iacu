# 穴道定位技術使用擴增實境實現於智慧型手機(iacu)

## 介紹
中醫針灸與穴位按摩是中醫常用的療法，根據病患不同症狀可以針對對應的穴位做針灸或按摩來緩解病患症狀。由於有龐大的穴位點數量，沒有受過專業訓練的人很難記得每個穴位的位置及對應的治療用途。系統藉由擴真實境，穴位會顯示在人體的輸入影像上，相比傳統使用穴位點量測電阻值，透過軟體的方式，使用已有的智慧型手機，在不 需要額外的硬體支出的情況下，能夠幫助病人快速定位穴位點用於按摩達到緩解輕微症狀的功效。
